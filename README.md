## About The Project

This project is created to test, demonstrate and analyze implementations of digital ID technologies via blockchain.

In recent years, digital identification solutions have become increasingly in demand. Therefore, this project is being created for a deep study of digital ID technologies.

It is supposed to search for and demonstrate vulnerabilities in order to make suggestions for improvement. Also, with the help of this project, it is planned to explore the possibilities of standardization and commercialization of the above technologies.

## Usage

Use HTTP POST request with `localhost:8081/block` header to add block in chain.

Use HTTP GET request with `localhost:8081/chain` header to get full chain in json.

You can also use ports :8082 and :8083
